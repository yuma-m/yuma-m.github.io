Awesome Transpose

2021/01/24 2-minute read

この記事では、ポップスにおけるサビへの転調パターンと主に J-POP の有名曲での採用例を紹介します。思いつき次第随時追記します。

真裏のキーとかなんぞやという方はこちらを先にご参照ください: 五度圏 - Wikipedia

Key +6 (6b/6#)

転調の中で一番難しいのではないかと思われる、真裏のキーへの転調です。

ももいろクローバー - 行くぜっ！怪盗少女

  • Key: Am -> Ebm
  • コード進行: IIm7-5 -> V7 -> Vbm
  • https://youtu.be/u7z9M0vFPbI?t=49

SHISHAMO - 明日も

  • Key: C# -> G
  • コード進行: V -> VIb -> VIIb -> Vb
  • https://youtu.be/zhCtzmDWsN0?t=82

Key +5 (1b)

下属調への転調です。C だったら F への転調。

YOASOBI - 怪物

B メロからサビへの転調です。

  • Key: Abm -> Dbm
  • コード進行: VI -> V -> IIb
  • https://youtu.be/dy90tA3TT1c?t=33

Key +4 (1b)

YOASOBI - 怪物

落ちサビから大サビへの転調です。

  • Key: Bbm -> Dm
  • コード進行: VI -> VII -> I
  • https://youtu.be/dy90tA3TT1c?t=149

Key +3 (3b)

音楽理論的に綺麗で、よく使われるパターンです。同主調の平行調。

スキマスイッチ - 奏(かなで)

大サビへの転調パターン。厳密には大サビ前の C メロで転調しています。

  • Key: Bb -> Db
  • コード進行: VIm7 -> VIbM7
  • https://youtu.be/J5Z7tIq7bco?t=206

ブラックビスケッツ - Timing

サビへの転調パターン、こちらは短調です。

  • Key: Cm -> Ebm
  • コード進行: Vm -> V -> IIIm (Im)
  • https://youtu.be/e_NcSaGkuNA?t=60

YOASOBI - 夜に駆ける

落ちサビから大サビへの転調パターン。

  • Key: D -> F
  • コード進行: IV -> III -> Vb -> V -> VIb
  • https://youtu.be/x8VYWazR5mE?t=203

Key +2 (2#)

いわゆる全音上げ。サビで盛り上げるパターンや、大サビなどで使われます。

B’z - ZERO

  • Key: Am -> Bm
  • コード進行: Im -> VI# (V) -> IIm (Im)
  • https://youtu.be/TUmni8kmlN4?t=130

YUKI - ビスケット

  • Key: E -> F#
  • コード進行: V -> VI (V) -> II (I)
  • https://youtu.be/-tVvtzn0B34?t=82

Key +1 (5b/7#)

いわゆる半音上げ。こちらもサビで盛り上げるパターンや大サビで多用されています。

MISIA - Everything

  • Key: Db -> D
  • コード進行: IIm7/V -> IIIbm7/VIb -> IIb
  • https://youtu.be/aHIR33pOUv0?t=333

Key -1 (7b/5#)

半音下げです。サビに向けての転調で使われることは稀。

YOASOBI - 夜に駆ける

落ちサビへの転調で採用。ちなみに落ちサビから大サビへは +3 です。

  • Key: Eb -> D
  • コード進行: IV -> III7 -> III
  • https://youtu.be/x8VYWazR5mE?t=186

B’z - 恋心 (KOI-GOKORO)

サビへの転調で利用。

  • Key: C -> B
  • コード進行: II -> V -> III -> IV# -> IIIM7
  • https://youtu.be/Use6JCncfyc?t=58

Official髭男dism - Cry Baby

サビの中で転調。メロディが続いている中での転調は非常に珍しいです。

  • Key: D -> Db
  • コード進行: V -> VI -> I/II -> IV
  • https://youtu.be/O1bhZgkC4Gw?t=54

Key -2 (2b)

J-POP ではサビで全音下がるようなパターンが全然見つけられず。。。

Beatles - Penny Lane

  • Key: B -> A
  • コード進行: V -> IV (V) -> VIIb (I)
  • https://youtu.be/z7syIxQCquo?t=30

Key -3 (3#)

+3 同様に音楽理論的に綺麗で、よく使われるパターンです。平行調の同主調。

シャ乱 Q - シングルベッド

  • Key: E -> Db
  • コード進行: IV -> IIm7 -> III (V) -> VIIb (I)
  • https://youtu.be/UYXNA54dDKs?t=88

Superfly - 輝く月のように

  • Key: C -> A
  • コード進行: II -> III (V) -> VI (I)
  • https://youtu.be/gG7evVU0OdA?t=94

渡辺美里 - My Revolution

  • Key: B -> Ab
  • コード進行: IV -> V -> III (V) -> VI (I)
  • https://youtu.be/oOAum6HWX-o?t=65

Key -4 (4b)

意外と少ない、C だったら Ab への転調。

嵐 - Hapiness

  • Key: F -> Db
  • コード進行: IV -> II -> V -> VIb -> IIb
  • https://youtu.be/HeXVJvEYynw?t=75

Key -5 (1#)

属調への転調です。C だったら G への転調。

椎名林檎 & トータス松本 - 目抜き通り

  • Key: Eb -> Bb
  • コード進行: IIm -> IIIm -> V
  • https://youtu.be/K6ptsqlfJZ8?t=165
